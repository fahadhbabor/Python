# 🐍 Python Learning Journey

This repository documents my journey of learning Python, covering fundamental concepts, hands-on practice, and small examples to build a strong programming foundation.

## 📚 Topics Covered

### 1. Hello, Python
A quick introduction to Python syntax, variable assignment, and working with numbers.

### 2. Functions and Getting Help
Understanding how to call functions, define custom functions, and use Python’s built-in documentation.

### 3. Booleans and Conditionals
Using boolean values and conditional statements to control program flow.

### 4. Lists
Working with lists, including indexing, slicing, and modifying data.

### 5. Loops and List Comprehensions
Using `for` and `while` loops, along with Python’s powerful list comprehensions.

### 6. Strings and Dictionaries
Exploring string manipulation and dictionary data structures.

### 7. Working with External Libraries
Learning how to import and use external libraries, along with best practices.

## 🎯 Goal
To build a solid foundation in Python and gradually move toward more advanced topics like data science and machine learning.
